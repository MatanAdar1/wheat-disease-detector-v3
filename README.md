# 🌾 מערכת ניטור חיישנים ואבחון מחלות חיטה (v3)
### Wheat Disease Detector & Greenhouse Sensor Monitoring System

מערכת מתקדמת לניטור חיישני חממה, אבחון חזותי של מחלות בעלי חיטה באמצעות למידה עמוקה (Deep Learning), וניהול נתונים בזמן אמת המבוססת על ענן **MongoDB Atlas**.

* **מבצעים:** נבו הלר ומתן אדר
* **מנחה:** אסי ברק

---

## 🚀 תכונות מרכזיות

האפליקציה מחולקת ל-3 מודולים עיקריים:

1. **📸 אבחון חזותי מהיר (ResNet18):**
   * העלאת תמונת עלה מתוך קובץ או צילום ישיר מהמצלמה.
   * סיווג מבוסס מודל **ResNet18** ל-5 מחלות/מצבים:
     * **BlackPoint** (חוד שחור)
     * **FusariumFootRot** (ריקבון בסיס הקנה)
     * **HealthyLeaf** (עלה בריא)
     * **LeafBlight** (קמלת עלים)
     * **WheatBlast** (פיריקורליית החיטה)
   * הצגת תוצאות אבחון, סף ביטחון (Confidence), תיאור המחלה והמלצות טיפול חקלאיות.

2. **📊 ניטור חיישנים ומעקב ניסוי:**
   * דפדפוף אינטראקטיבי בין צמחי החיטה השונים בניסוי.
   * הצגת מדדי לחות קרקע, טמפרטורה וקבוצת טיפול (Drought / Control).
   * גרפי סדרות זמן (Time-Series) דינמיים של מדדי לחות וטמפרטורה הנמשכים מ-MongoDB.
   * שיוך אבחונים חזותיים והערות חממה לפי מזהה צמח (`plant_id`).
   * צפייה בהיסטוריית אבחונים מלאה ואפשרות מחיקה.

3. **⚙️ ניהול נתונים וסנכרון ענן:**
   * העלאת קובצי חיישנים מעודכנים בפורמט Excel (`.xlsx`) או CSV (`.csv`).
   * עיבוד אוטומטי של עמודות החיישנים ועדכון ה-Database ב-MongoDB Atlas.
   * הצגת מטריקות סטטוס בזמן אמת של בסיס הנתונים (כמות צמחים, מדידות ואבחונים).

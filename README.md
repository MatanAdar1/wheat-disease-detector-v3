# Wheat Disease & Greenhouse Sensor System v3 🌾

מערכת מתקדמת לזיהוי מחלות חיטה ואיסוף נתוני חיישני חממה (לחות וטמפרטורה) בזמן אמת.

## 🚀 תכונות עיקריות בגרסה v3
* **טעינת מודל ישירה:** מודל ה-ResNet18 (`best_resnet18_wheat.pt`) מאוחסן ישירות ב-Git.
* **ניטור חיישנים בזמן אמת:** תמיכה מלאה בסדרות זמן של 3,604 מדידות לחות וטמפרטורה ל-35 צמחי חיטה.
* **גרפים דינמיים:** תצוגת מגמות לחות וטמפרטורה בלחיצת כפתור.
* **ענן MongoDB Atlas:** אגירה וסינכרון של נתוני הצמחים, המדידות והיסטוריית האבחונים.
